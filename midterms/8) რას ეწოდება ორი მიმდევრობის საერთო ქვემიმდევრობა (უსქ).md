# მოიყვანეთ უსქ-ის ალგორითის ფსევდოკოდი და შეაფაასეთ ალგორითმისთვის საჭირო ოპერაციების რაოდენობა

```
LCS-LENGTH(X, Y) :
  m = len(X);
  n = len(Y);
  for i=0; i<=m; i++ :
    c[i][0] = 0;
  for j=1; j<=n; j++ :
    c[0][j] = 0;
  for i=1; i<=m; i++ :
    for j=1; j<=n; j++ :
      if X[i] == Y[i] :
        c[i][j] = c[i-1][j-1] + 1;
        b[i][j] = "upLeft";
      elif c[i-1][j] >= c[i][j-1] :
        c[i][j] = c[i-1][j];
        b[i][j] = "up";
      else:
        c[i][j] = c[i][j-1];
        b[i][j] = "left";
  return c,b;
```

ოპერაციების შესრულების დროა O(m*n)
